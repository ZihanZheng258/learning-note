目前为止，我们只使用了单个组件。真正的 Vue 应用往往是由嵌套组件创建的。

父组件可以在模板中渲染另一个组件作为子组件。要使用子组件，我们需要先导入它：

```JavaScript
import ChildComp from './ChildComp.vue'
```
然后我们就可以在模板中使用组件，就像这样：

```JavaScript
<ChildComp />
```
现在自己尝试一下——导入子组件并在模板中渲染它。

```JavaScript
<script setup>
import ChildComp from './ChildComp.vue'
</script>

<template>
  <ChildComp />
</template>
```
