汇编过程将上一步的**汇编代码**(`main.s`)**转换成机器码**(machine code)，**这一步产生的文件叫做目标文件**(`main.o`)，**是二进制格式**。

gcc/g++的汇编过程通过 `as` 命令完成，所以我们可以通过`g++ -c`或`as`命令完成汇编

```bash
g++ -c -I include/ main.cpp -o main.o
# 或者直接调用 as 命令
as main.s -o main.o
```

```
上述指令中： 
- `g++ -c`让编译器在汇编之后退出，等价于 `as` 
- `-I include/`仍是用于指定头文件目录 
- `main.cpp`是要汇编的源文件 
- `-o main.o`用于指定生成的文件名
```
